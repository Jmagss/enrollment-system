    Planning

    Having an effective and efficient enrollment system is a plus to a school whereas the faculty and staff would benefit together with students. In this phase the researcher plan on how to create and what is needed in the system.

    Analysis

    In analysis moreaccentuation is given to understanding the points of interest of an existing study or a proposed one and after that choosing whether the proposed study is alluring or not and whether the existing framework needs enhancements.

    System Design

    We ensure that the design of the said system is suitable and easy to recognize in putting in a simple but detailed way of design.

    Development

    In this phase the proponents develop the system.

    Testing

    This phase includes the process of evaluating a system or application, in order to check whether the application meets all requirements of the client and to detect the error. Requirements such as software functions and features where checked to determine if it will work according to the specification.

    Implementation

    After the proponents develop the system, they need to install it so that the user will be familiar to use the system. The users of the system will check if the recommended functions and suggestions of the user are meet.

    Maintenance

    Every system is human-like, it gets tired, it will be slow, and even give up when the user is overuse, that is why a good maintenance of a system is needed.

    Computerized Enrollment System for Mandaue City College.
